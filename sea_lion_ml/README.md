# 🦁 Sea Lion Population Trend Classification
### MSIS 522 — Analytics & Machine Learning | University of Washington

An end-to-end machine learning pipeline for predicting whether a Steller sea lion survey site shows a **declining population trend**, using ~50 years of count data from NOAA/NMFS aerial surveys across Alaska and the US Pacific coast.

---

## 📋 Project Overview

| | |
|---|---|
| **Task** | Binary Classification (Declining vs Stable/Recovering) |
| **Data** | NOAA ALLCOUNTS survey data (4 sheets: WDPS non-pup, WDPS pup, EDPS non-pup, EDPS pup) |
| **Sites** | 512 survey sites spanning 1970–2024 |
| **Models** | Logistic Regression, LASSO, Ridge, CART, Random Forest, LightGBM, MLP |
| **CV** | 5-fold stratified cross-validation + GridSearchCV |
| **Explainability** | SHAP (tree models) + logistic coefficients |
| **Clustering** | KMeans (k=4) + PCA |

## 🚀 Quick Start

### 1. Clone the repository
```bash
git clone https://github.com/burlynb/MSIS-SSL-CountModels
cd sea-lion-ml
```

### 2. Install dependencies
```bash
pip install -r requirements.txt
```

### 3. Train models (run once)
```bash
python train.py
```
This will process the data, train all models with cross-validation, compute SHAP values, and save everything to the `models/` directory.

### 4. Launch the Streamlit app
```bash
streamlit run app.py
```

## 🔬 Feature Engineering

Features are computed per survey site from the time-series of annual counts:

| Feature | Description |
|---------|-------------|
| `mean_count` | Average count across all survey years |
| `recent_mean` | Mean of last 5 observed counts |
| `early_mean` | Mean of first 5 observed counts |
| `pct_decline_from_peak` | % loss from historical maximum to recent mean |
| `cv` | Coefficient of variation |
| `n_years` | Number of survey years |
| `r_squared` | R² of linear trend fit |
| `trend_pct_per_year` | Average % change per year |
| `years_since_peak` | Years since maximum count |
| `dps` | WDPS (endangered) or EDPS (recovering) |
| `region` | Geographic sub-region |
| `count_type` | Non-Pup (adults) or Pup (juveniles) |

**Target**: `is_declining = 1` if linear slope < 0 AND recent/early ratio < 0.85
